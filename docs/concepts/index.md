# Basics
Decapod는 Kubernetes Cluster 환경을 기반으로 하며 Argo Workflow, Argo CD, Kustomize 등의 오픈소스 S/W를 활용하고 있다.  
Decapod를 사용하기 앞서, 각 프로젝트에 대한 기본적인 이해를 하면 Decapod에 대해 더 쉽게 이해할 수 있다.
